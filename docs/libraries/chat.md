# @adopt-dont-shop/lib.chat

Real-time chat and messaging built on Socket.IO. Provides `ChatService`, the `useConnectionStatus` hook, and admin React Query hooks (`useAdminChats`, `useAdminChatMessages`, etc.).

The canonical, code-verified reference lives next to the source:

> **[lib.chat/README.md](../../lib.chat/README.md)**

It covers exports, `ChatService` configuration, message / typing / connection events, reconnection options, and the React hook surface.
