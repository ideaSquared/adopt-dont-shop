# @adopt-dont-shop/lib.discovery

## Purpose

Swipe-based pet-discovery client for the frontend apps — fetches the discovery
queue, records like/pass/super-like actions, and manages swipe sessions.

## Location in the architecture

See [`docs/README.md`](../../docs/README.md#libraries) for where the shared
libraries sit. Reaches the gateway's discovery surface over
[`service.matching`](../../services/matching/README.md), and consumes the
match-pick types from [`lib.matching`](../lib.matching/README.md).

## Public API / exports

The canonical list lives in [`src/index.ts`](src/index.ts):

- `DiscoveryService` — constructed with `DiscoveryServiceConfig`:
  `getDiscoveryQueue`, `loadMorePets`, `recordSwipeAction`, `getSwipeStats`,
  `startSwipeSession` / `endSwipeSession`, `getSessionStats`.
- Types: `DiscoveryServiceConfig`, plus the pet / session / action types
  re-exported from `./types`; endpoint constants from `./constants`.
