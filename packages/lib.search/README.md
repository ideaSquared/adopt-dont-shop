# @adopt-dont-shop/lib.search

## Purpose

Cross-domain search client for the frontend apps — pets, messages, faceted
search, and typeahead suggestions.

## Location in the architecture

See [`docs/README.md`](../../docs/README.md#libraries) for where the shared
libraries sit. Reaches the gateway's search surface (backed by
[`service.matching`](../../services/matching/README.md), which absorbed the
search module).

## Public API / exports

The canonical list lives in [`src/index.ts`](src/index.ts):
