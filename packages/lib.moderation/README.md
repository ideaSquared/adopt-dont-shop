# @adopt-dont-shop/lib.moderation

## Purpose

Content moderation and reporting for the frontend apps: Zod-validated report
models, an API service for the moderation endpoints, React hooks for fetching /
managing moderation data, and display formatting utilities.

## Location in the architecture

See [`docs/README.md`](../../docs/README.md#libraries) for where the shared
libraries sit. Consumed by the admin moderation surface; the backing endpoints
are the gateway's `/api/v1/admin/moderation/*` routes over
[`service.moderation`](../../services/moderation/README.md).

The canonical list lives in [`src/index.ts`](src/index.ts):
