# @adopt-dont-shop/lib.applications

## Purpose

Adoption-application lifecycle client for the frontend apps — submit, update,
transition stage, and manage supporting documents, for both applicants and
rescue staff.

## Location in the architecture

See [`docs/README.md`](../../docs/README.md#libraries) for where the shared
libraries sit. Reaches the gateway's `/api/v1/applications/*` +
`/api/v1/application-documents/*` surface over
[`service.applications`](../../services/applications/README.md).

## Public API / exports

The canonical list lives in [`src/index.ts`](src/index.ts):
