# @adopt-dont-shop/lib.support-tickets

## Purpose

Customer-support ticket management for the frontend apps: Zod schemas, a service
client, React Query hooks, and display utilities.

## Location in the architecture

See [`docs/README.md`](../../docs/README.md#libraries) for where the shared
libraries sit. Reaches the gateway's `/api/v1/admin/support/*` +
`/api/v1/support/*` surface over
[`service.moderation`](../../services/moderation/README.md), which owns the
support-ticket tables.

## Public API / exports

The canonical list lives in [`src/index.ts`](src/index.ts):

- `SupportTicketService` + `supportTicketService` — client over the support

  `updateTicket`, `assignTicket`, `addResponse`, `escalateTicket`,
  `close/resolve/reopenTicket`, `setPriority`, `rateTicket`, …).
